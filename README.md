# geekhub2019
Cute Taxi
=========

The Taxi application with simple navigation

In app is used such libraries:
[DB Room][1],[Retrofit][2], [OkHttp][3], [Moshi][4], [MVVM-pattern][5];
and tests: [Espresso][6], [Mockito][7], Hamcrest, JUnit and other
